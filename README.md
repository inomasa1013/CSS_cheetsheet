## 環境設定 !!

プロジェクトをクローン！

```
$ git clone https://github.com/*username/*yourProject.git
$ cd *yourProject/front
```

お次は必要なモジュールをインストール！

```
$ cd front
$ npm install
$ cd ..
$ cd back
$ npm install
```

### 使い方ッ !!

**back フォルダにディレクトリを移動してね！**

```
$ cd back
```

_npm start でサーバー起動してね！_

```
$ npm start
```
